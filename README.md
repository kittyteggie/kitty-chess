# (•ω•) Kitty Chess  

A lightweight chess game coded completely in Python with Stockfish as a CPU and that saves all your games as [.pgn files](https://en.wikipedia.org/wiki/Portable_Game_Notation).  

---
## Dependencies install  
You must install these dependencies for playing the game:  
```pip install pygame``` (```pip3 install pygame``` in some systems)  
  
Package: ```stockfish``` (full command depends of your Linux distro/package manager)
